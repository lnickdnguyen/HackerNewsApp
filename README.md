# Hacker News App

Full-stack app using Angular + .NET Core + Hacker News API

## Structure

- `client/` – Angular frontend
- `server/` – .NET Core backend

## Running Locally

### Backend

```bash
cd server
dotnet run
```

### Frontend

```bash
cd client
npm install
ng serve --open
```

## Running unit tests

```bash
ng test
```
